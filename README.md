# d3-topo

> 基于D3-Force力导向图实现和封装的拓扑图绘制工具  

## 用法  
npm install d3-topo

## 功能  
1. 展示节点之间的连线关系  
2. 支持缩放、平移和双击重置功能  
3. 提供节点、链路、空白处的点击事件，供外部自定义交互效果  
4. 支持扩展工具栏和小地图  

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
